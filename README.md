# MAGA-English
[A project by HSE Group 53 for the "MVP in DS" course (2026)]

A Telegram bot that allows you to learn and practice English by communicating with a digital copy of Donald Trump. The bot imitates his unique speech patterns: vocabulary, intonation in written form, characteristic interjections, and repetitions.

# Описание проекта

## 1. Название проекта (рабочие варианты)
*   Английский с Дональдом
*   Trump English
*   English with Donald
*   MAGA English
*   Trump Talks

## 2. Целевая аудитория
1.  **Изучающие английский:** люди с уровнем pre-intermediate и выше, которым скучно учить язык по учебникам, но интересна поп-культура и политика.
2.  **Фанаты или хейтеры Трампа:** те, кому интересно просто пообщаться с «Трампом» ради развлечения и мемов, а в процессе — невольно практиковать язык.
3.  **Криэйторы,** которые ищут необычный контент для реакций или стримов.

## 3. MVP — что делаем в первую очередь
**Цель MVP:** проверить гипотезу, что пользователям интересен формат общения в формате текстового чата с персонажем для изучения языка.

**Функционал MVP:**
1.  Стартовое меню с кнопками «Начать урок» и «Поболтать».
2.  **Чат с контекстом:**
    *   пользователь пишет сообщение;
    *   бот отвечает в стиле Трампа;
    *   системный промпт для нейросети прописан максимально жёстко. Упрощённый пример: «Ты — Дональд Трамп. Отвечай короткими, эмоциональными фразами. Используй повторы. Начинай предложения с look, listen. Добавляй междометия „э-э“. Никогда не извиняйся. Ты лучший учитель английского». Вариант полного промпта см. Приложение 2.
3.  **Сохранение контекста:** бот помнит последние 5–10 сообщений, чтобы диалог был связным.
4.  **Ограничение по количеству запросов** в день, чтобы не разориться на API.

**Стек технологий для MVP:**
*   **платформа:** telegram bot API (pyTelegramBotAPI / aiogram на python);
*   **нейросеть:** на старте можно использовать chatgpt через OpenAI API с мощным системным промптом. Альтернативы — claude или один из российских провайдеров, если важна оплата в рублях;
*   **база данных:** redis или postgresql для хранения истории диалога и идентификаторов пользователей.

## 4. Этапы развития
**Фаза 1: MVP (запуск)**
*   один персонаж — Трамп;
*   только свободный диалог;
*   монетизация через рекламу и партнёрские ссылки на курсы английского.

**Фаза 2: улучшение стиля**
*   сбор датасета: парсинг стенограмм речей, интервью, дебатов;
*   fine-tuning open source модели на этих данных, что сделает ответы более аутентичными.

**Фаза 3: расширение функционала (премиум)**
*   сценарные уроки — заранее прописанная канва, от которой бот не отходит;
*   интерактивное тестирование по лексике из диалога;
*   добавление других персонажей с возможностью выбора в меню.

## 5. Монетизация
1.  **Freemium-модель:** 20–30 бесплатных сообщений в день, далее платная подписка за безлимит и доступ к урокам.
2.  **Нативная реклама** в ответах бота со ссылками на партнёрские курсы.
3.  **Микротранзакции,** например «подарок» персонажу, который разблокирует особый стиль общения.

## 6. Риски и их решение
1.  **Юридический риск** использования образа реального человека. Решение на mvp — позиционировать проект как пародию и образовательный инструмент. При масштабировании привлечь юриста по интеллектуальным правам.
2.  **Технический риск** высоких расходов на API. Решение — жёсткие лимиты, оптимизация промптов, переход на свои fine-tuned модели.
3.  **Этический риск:** бот может сказать нечто обидное или спорное. Решение — добавить фильтр цензуры на входе и выходе.

## 7. Метрики успеха
*   возвращаемость пользователей;
*   среднее количество сообщений в одном сеансе;
*   доля пользователей, купивших подписку.
